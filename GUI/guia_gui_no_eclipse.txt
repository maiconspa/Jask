criando um novo projeto de interface gráfica no Eclipse:

File > New > Java project:
    insira o nome do projeto e 'finish'.

-----
Criando o arquivo para confecção de telas:

acesse o 'workbench', vá no diretório 'src' do seu projeto, crie um pacote,
no nosso caso, um nome bem sugestivo seria "view".

após, clicar com o botão direito sobre o pacote: New > Other > WindowBuilder > Swing Designer > Application Window > 'insira um nome para a tela' e clique em 'finish'

*se o 'WindowBuilder' não aparecer em 'Other', repita o processo de instalação do WindowBuilder*

feito isso você verá duas abas na parte inferior do Eclipse, são elas a Source (código bruto) e a Design, esta ultima que por sua vez permite visualizar, criar e alterar aspectos de design da interface.
