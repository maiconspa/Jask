DOWNLOAD GIT CONSOLE: https://git-scm.com/downloads


verificar versão do git:
    git --version


login (troca o "YOUR_USERNAME" pelo seu nomeDeUsuario ):
    git config --global user.name "YOUR_USERNAME"


verifique se o nomeDeUsuario está correto:
    git config --global user.name


Entre com o seu e-mail:
    git config --global user.email "your_email_address@example.com"


verifique se o email está correto:
    git config --global user.email


verificar lista de dados de login:
    git config --global --list


inicializar diretório local de controle de versão:
    git init


CLONAR REPOSITÓRIO:
    git clone https://github.com/username/repositoryName


SUBMETER ALTERAÇÕES:

*selecionar arquivos para "monitorar" (subir para o repositório, comentar...)
    git add arquivo
    ou se desejar adicionar todos de uma vez: git add *

verificar estado de arquivos (mostra se há arquivos modificados)
    git status

comentar alterações:
    git commit -m "tente descrever as alterações sempre pensando em: isso serve para..."
